Movie Webpage Project

Context

For my SEP Movie Project, I needed to work with a partner and use our knowledge of basic HTML and CSS to create a fake movie webpage. My partner decided to make a lego-themed action/horror movie, and the title was "Captain Ranger's Last Stand", while I did all of the styling

Content

To start our project, we needed to make a plan. We planned it out pretty concisely. Since my partner didn't really know any CSS, he was tasked with writing the story and adding the accompanying images, while I had to style the story. For the HTML part of the movie, he needed to include

  • headings
  • paragraph
  • two local images
  • inputs and submit button

    • For the CSS part, I had to add

  • element and class and id selectors
  • divs and spans to subsequently style my page
  • imported fonts, which needed to be mentioned in CSS
  • border and padding and margin properties

    • Reflection


    Challenges

    I greatly struggled with spacing and positioning throughout this project. When adding my linear gradient background image, there was white space all around the entire webpage. I had to set the top, bottom, left and right margin properties for the linear-gradient, and after this still produced some white space on top of the webpage or to the side, I had to set the margin properties, respectively, for each element within a div-with-no-border. Finally, there was still white spacing to the right of all of the content of the webpage which was unaffected by any margin properties. This was caused by the size of the images, and making them smaller solved the issue. Another CSS property, overflow-x, with its value set to 'hidden' prevents the user from scrolling along the x-axis and revealing the white space, but subsequently, cuts off the images.

    Takeaways

    I learned how to manipulate the padding property of an element (for example, h1) inside of a CSS element such as .div-with no-border --that div-with-no-border being the class value of the html element that is inside of it in html-- to style a webpage exactly how I want it to look; making completely circlular elements on my webpage be the right size and shape to comfortably fit elements inside with the use of the padding property.

    I need to always pay attention to the size of the image (width)

    Next Step

    What I struggled with most was spacing and positioning so I need to practice this in the future. The only way I found to add make an image that is too big not cause the overflow of white space to the left of the entire webpage, was to add the 'overflow-x' property to the body and html elements in CSS, buts this cuts off part of the image. In the future, I need to find a way to make the image as big as I want it to be without it flowing outside of the background image; without obstructing the image itself in any way.

    Code used to style my webpage when my image was to large
