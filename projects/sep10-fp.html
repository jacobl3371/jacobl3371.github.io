SEP10 Freedom Project

Context

The Freedom Project is a course long project wherein students combine all of the knowledge they learned throughout the course, in the case of SEP 10, this entails using the knowledge of HTML and CSS (basic web design) and Bootstrap (advanced web design) as well as a software tool of my choosing to create a website which addresses the currently existing and future technologies in an industry of my choosing.

I chose the industry of education, discussing the currently existing online educational softwares used and the issue behind this industry, which is that these services can not be accessed psuedononymously and seperate from the rest of the data on your operating system and devices, with the use of the only software which does this, the Tor browser which utilizes the anonymity network Tor to encrypt and anonymize all traffic. The browser is isolationary and amnesic so as soon as it is closed out, all cookies are cleared and none of the sessions are associated with each other. Along with the use of the Tor network which selects 3 random IP addresses running relays which facilitate the Tor network and encrypts and anonymizes the user's traffic through these three IP addresses. I tried to see how much agency a K-12 student could attain over the data collected on them daily through the use of online educational services, and used my research to build my Freedom Project website.
